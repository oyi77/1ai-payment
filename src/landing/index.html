Why 1ai-payment?

🔌

Single API

One integration powers 11 gateways. switch providers without changing code — just change the gateway parameter.

🔄

Async Webhook Routing

All gateways normalize to a uniform callback format. Forward events to your project with automatic retry and dead letter queue.

🔑

Merchant-Scoped Auth

Each merchant gets an API key. Rate limits, refunds, and transaction history isolated per merchant.

📊

Merchant Dashboard

Track transactions, manage refunds, configure gateways, and rotate API keys — all from a web portal.

📕

OpenAPI Docs

Full Swagger UI auto-generated from code. Try every endpoint directly from the browser.

Signature Verification

Every gateway webhook is verified with timing-safe comparison. Idempotency enforced at the database level.
